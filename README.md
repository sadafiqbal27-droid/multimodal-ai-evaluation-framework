# LLM Evaluation Framework

A modular Python framework for evaluating Large Language Model (LLM) responses across instruction following, reasoning quality, hallucination detection, safety, and overall quality assessment.

## Project Goal

This project aims to automate the evaluation of LLM-generated responses using structured scoring criteria inspired by real-world AI evaluation workflows.

## Planned Features

- Instruction Following Evaluation
- Reasoning Quality Assessment
- Hallucination Detection
- Safety Evaluation
- Rubric-Based Scoring
- Overall Quality Score
- JSON Report Generation
- Streamlit Dashboard

## Technology Stack

- Python
- JSON
